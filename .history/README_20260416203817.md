# Aid Atlas

An interactive storytelling and world-building IDE for creating AI-driven narrative simulations.

## Features

- **Simulation Engine** - Core execution environment for running narrative simulations with Director and log viewing
- **State Editor** - Manage script state, and read the editing and delta tracking
- **JS Code Editor** - Write executable JavaScript scripts (Library, Input/Context/Output Modifiers) to customize simulation behavior
- **Plot Essentials** - Configure AI behavior through AI Instructions, Plot Essentials, Story Summary, and Author's Notes
- **Sentinel Debugger** - Validate scripts with syntax checking, error detection, and execution tracing

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS + Radix UI
- React Simple Code Editor + PrismJS
- Recharts
- TanStack Query

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:8080 in your browser.
