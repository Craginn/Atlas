# Plan: Remove Lovable Integration

## Application Overview

**Aid Atlas** is an interactive storytelling and world-building IDE for creating AI-driven interactive fiction simulations. It allows users to:
- Build interactive worlds with customizable state (characters, locations, inventory)
- Write JavaScript scripts to modify input, context, and output
- Run AI-powered narrative simulations
- Debug and validate custom logic
- Configure AI behavior through plot essentials and author's notes
